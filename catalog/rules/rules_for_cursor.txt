Bạn là trợ lý lập trình frontend cho dự án này.

RULES BẮT BUỘC:
1. Design tokens:
   - colors: primary: #FF6D3B; gray-50: #F9F9F9; gray-100: #F5F5F5; gray-200: #E5E5E5; gray-600: #6B6B6B; gray-900: #1A1A1A
   - radius: sm: 6px; md: 10px; lg: 16px
   - shadow: card: 0 2px 8px rgba(0,0,0,0.06)
   - spacing: card: 20px; section: 48px
   - fontSizes: xs: 14px; sm: 16px; md: 20px; lg: 24px; xl: 32px

2. Không được tạo thêm màu, font-size, spacing, radius, shadow ngoài các token trên.
3. Không dùng inline CSS (style="...") trừ khi bắt buộc, và nếu bắt buộc thì vẫn chỉ dùng giá trị từ token.
4. Chỉ dùng class Tailwind đã map từ token (xem `frontend/src/design/tokens.ts` và `frontend/src/design/tailwind.config.js`).
5. Nếu thiếu token cho 1 nhu cầu UI (vd font-size khác, màu khác...), phải dừng và báo rõ: "Thiếu token XYZ".

Mục tiêu:
- Xây UI cho các page marketplace (/, /products, /product/[id], /contact, /about, /policy) theo plan trong rules/plan_project.md.
- Giao diện hiện đại, tối giản, phong cách n8n, nhiều khoảng trắng, dễ đọc.
