🎯 LUỒNG ỨNG DỤNG (Use Cases)
════════════════════════════════════════════════════════════════════════════════

1. QueryLeaveBalance
   Mục đích: Lấy số ngày phép còn lại
   Input: user_id, target_user_id (optional)
   Output: leave_balance, employee_name
   RBAC: ✅ Employee/Manager/Admin
   Status: ✅ HOÀN THÀNH

2. CreateLeaveRequest
   Mục đích: Tạo đơn xin phép
   Input: start_date, end_date, reason
   Validations: date range, not in past, sufficient balance
   Output: leave_request_id, status
   RBAC: ✅ All employees
   Status: ✅ HOÀN THÀNH

3. QueryLeaveRequests ⭐ NEW
   Mục đích: Lấy danh sách đơn xin phép
   Input: status (optional), limit, offset
   Filtering: By status, by employee
   Output: List of leave requests
   RBAC: ✅ Own/Team/All based on role
   Status: ✅ HOÀN THÀNH MỚI

4. ApproveLeave
   Mục đích: Duyệt đơn xin phép
   Input: leave_request_id
   Validation: Must be pending, only manager/admin
   Output: Updated status
   RBAC: ✅ Manager/Admin only
   Status: ✅ HOÀN THÀNH

5. RejectLeave ⭐ NEW
   Mục đích: Từ chối đơn xin phép
   Input: leave_request_id, rejection_reason (optional)
   Validation: Must be pending, only manager/admin
   Output: Updated status, rejection_reason
   RBAC: ✅ Manager/Admin only
   Status: ✅ HOÀN THÀNH MỚI
