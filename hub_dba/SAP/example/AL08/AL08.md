### Bức ảnh bạn gửi thuộc về T-code AL08 trong SAP.

### Thông tin chi tiết:
- **T-code: AL08 (System-Wide List of User Sessions)**.
- Khác với SM04 (chỉ xem user ở server hiện tại), AL08 cho phép bạn xem toàn bộ người dùng đang đăng nhập trên tất cả các Application Server của hệ thống SAP đó (ở đây là hệ thống EP0).
- **Nội dung bức ảnh nói gì**:
Đây là danh sách các phiên làm việc (sessions) của người dùng đang hoạt động trong hệ thống tại thời điểm chụp.
- **Các cột quan trọng**:
    - **User ID**: Tên tài khoản đang đăng nhập (ví dụ: SP1337, SP1478, SM1471...).
    - **Client**: Phân hệ (thường là 300).
    - **Application**: Các T-code hoặc chương trình mà người dùng đang mở (ví dụ: MB52, VF03, MIGO, MIRO...).
    - **Type of User Session**: Loại kết nối, thường là GUI (người dùng dùng phần mềm SAP Logon) hoặc RFC (kết nối giữa các hệ thống/ứng dụng).
    - **Memory (Net)**: Lượng bộ nhớ mà mỗi user đang chiếm dụng.
    - **Thống kê dưới cùng**: Hệ thống đang có 621 user sessions với 778 ABAP sessions (một user có thể mở nhiều cửa sổ/session cùng lúc).
- **Tóm lại**: Đây là màn hình quản trị hệ thống dùng để theo dõi xem ai đang làm gì, dùng T-code nào và chiếm bao nhiêu bộ nhớ trong toàn hệ thống SAP.
